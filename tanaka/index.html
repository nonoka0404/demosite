minorivraとは

ビジネスを学ぶプログラミングスクール

私たちが目指すのは、
「一般的なコーディングを教えるプログラミングスクール」
ではありません。

”プログラミングはあくまでツール”です。
プログラミングを用いて、どのように世の中の困りごとを解決していくのか、
どのように課題を解決していくのかというビジネスの思考を身につけます。

学べる言語

  • HTML

    Web制作では、ホームページ上に表示をしたい文章や写真などの情報に対し、HTMLという言葉で、コンピュータに文章構成を指示することから始まります。

  • CSS

    CSSは、主に見た目の装飾に関することを定義します。CSSとHTMLは常にセットで使用され、レイアウトや見た目の装飾が整い、見やすいデザインにします。

  • サイト設計

    Webサイトは、目的やターゲット、最終的なゴールなどに合わせた「サイト設計」を行なってはじめて、効果が現れます。

コース一覧

  • 基礎コース

    プログラミングの基礎知識がつき、自分で調べて解決する方法が身に付く。

  • 応用コース

    プログラミングの知識、課題解決力を身につける。社会課題への向き合い方を身につける。

  • フロントエンドコース

    フロントエンドのコーディング、wordpress化を学ぶ。

  • バックエンドコース

    GASやLarabelを用いたバックエンドのコーディングを学び、DX化やwebアプリ開発を進める実践的な技術を身につける。
